Multiple Select

Multiple Select 是一个通过复选框来选择多元素的 jQuery 插件:).

下载 开始使用

当前版本： v1.2.3

专为 HTML Select 标签设计

Multiple Select 通过简单的设置，就可以拥有强大的功能，大大提高工作效率，减少开发时间。

主要功能

  • 默认选项允许显示复选框。
  • 支持分组显示元素。
  • 支持在一行中显示多个选项。
  • 支持全选。
  • 支持显示默认提示信息。
