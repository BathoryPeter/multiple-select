选项

选项定义于 jQuery.fn.multipleSelect.defaults。

name

类型： String

说明： select 元素的 name 属性。

默认： ''

placeholder

类型： String

说明： 可以定义 placeholder 值，并在您选择项目之前显示 placeholder 值。

默认： ''

例子： The Placeholder

selectAll

类型： Boolean

说明： 显示选中所有复选框。

默认： true

例子： Hide Select All

single

类型： Boolean

说明： 允许您只选择一个选项。

默认： false

例子： Single Row

multiple

类型： Boolean

说明： 在一行中显示多个项目。

默认： false

例子： Multiple Items

hideOptgroupCheckboxes

类型： Boolean

说明： 隐藏 optgroup 复选框。

默认： false

例子： Hide Optgroup Check

multipleWidth

类型： Number

说明： 显示多个项目的宽度。

默认： 80

例子：

  • Multiple Items

    • width

    类型： undefined

    说明： 定义 select 的 width 属性，支持百分比设置。

    默认： undefined

    例子： Custom Dropdown Width

    dropWidth

    类型： undefined

    说明： 定义下拉框的宽度。

    默认： undefined

    例子： Custom Dropdown Width

    maxHeight

    类型： Number

    说明： 定义下拉列表的最大高度属性。

    默认： 250

    例子： Max Height

    position

    类型： String

    说明： 定义选择下拉列表的位置，只能是 'bottom' 或 'top'。

    默认： bottom

    例子： The Position

    displayValues

    类型： Boolean

    说明： 显示所选值而不是文本。

    默认： false

    例子： Display Values

    displayTitle

    类型： Boolean

    说明： 显示所选文本的 title。

    默认： false

    例子： Display Title

    displayDelimiter

    类型： String

    说明： 自定义显示分隔符。

    默认： ,

    例子： Display Delimiter

    minimumCountSelected

    类型： Number

    说明： 仅当选择了多于 minimumCountSelected 项时，才会显示 formatCountSelected。

    默认： 3

    例子： Minimum Count Selected

    ellipsis

    类型： Boolean

    说明： 如果设置了 minimumCountSelected，则在所选选项后添加 '...'。覆盖 formatCountSelected 选项。

    默认： false

    例子： The Ellipsis

    isOpen

    类型： Boolean

    说明： 默认情况下打开选择下拉列表。

    默认： false

    例子： Is Open

    keepOpen

    类型： Boolean

    说明： 始终打开选择下拉列表。

    默认： false

    例子： Keep Open

    openOnHover

    类型： Boolean

    说明： 通过鼠标经过而不是点击来打开选择下拉列表。

    默认： false

    例子： Open On Hover

    container

    类型： object

    说明： 定义自定义容器以解决显示错误问题，例如父节点是 overflow: hidden.

    默认： null

    例子： The Container

    filter

    类型： Boolean

    说明： 显示搜索字段以显示搜索框项。

    默认： false

    例子： Basic Filter Filter With Optgroup

    filterGroup

    类型： Boolean

    说明： 仅筛选 optgroups 而不筛选选项。

    默认： false

    例子： Filter Only Optgroup

    filterPlaceholder

    类型： String

    说明： 设置过滤器的 placeholder.

    默认： ``

    例子： Filter Placeholder

    filterAcceptOnEnter

    类型： Boolean

    说明： 加快键盘使用。输入过滤器文本然后按 Enter 或空格将自动单击全选并关闭下拉列表。

    默认： false

    例子： Filter Accept On Enter

    animate

    类型： undefined

    说明： 定义打开或关闭下拉列表的动画。可以是 undefined，'fade' 和 'slide'。

    默认： undefined

    例子： The Animate

    styler

    类型： function

    说明： 项目样式器函数，返回样式字符串以自定义项样式，包含参数： value。

    默认： { return false }

    例子： The Styler

    textTemplate

    类型： function

    说明： 项目 textTemplate 函数，返回字符串以自定义项目文本，包含参数：$elm。

    默认： { return $elm[0].innerHTML }

    例子： Text Template

    labelTemplate

    类型： function

    说明： 项目 labelTemplate 函数，返回自定义 optgroup 标签的字符串，包含参数：$elm。

    默认： { return $elm[0].getAttribute('label') }

    例子： Label Template

    事件

    使用事件语法为：

    $('#select').multipleSelect({
  onEventName: function (arg1, arg2, ...) {
    // ...
  }
})

    例子：The Event

    onOpen

    • 参数： -

    • 说明： 当下拉列表被打开时触发。

    onClose

    • 参数： -

    • 说明： 当下拉列表被关闭时触发。

    onCheckAll

    • 参数： -

    • 说明： 通过点击“全选”复选框或以编程方式调用“checkall”方法选中所有选项时触发。

    onUncheckAll

    • 参数： -

    • 说明： 通过点击“全选”复选框或以编程方式调用“uncheckall”方法取消选中所有选项时触发。

    onFocus

    • 参数： -

    • 说明： 将事件处理程序绑定到“focus”。

    onBlur

    • 参数： -

    • 说明： 将事件处理程序绑定到"blur"。

    onOptgroupClick

    • 参数： view

    • 说明： 点击optgroup标签时触发。view 参数是一个对象，它包含 label, checked, children。

    onClick

    • 参数： view

    • 说明： 选中或取消选中复选框时触发。view 参数是一个对象，它包含 label, value, checked。

    onFilter

    • 参数： text

    • 说明： Multiple Select 进行过滤的时候触发。

    onAfterCreate

    • 参数： -

    • 说明： Multiple Select 创建完成后触发。

    方法

    使用方法的语法：

    $('#multiple').multipleSelect('method', parameter)

    getSelects

    • 参数： type

    • 说明： 获取选定的值。type 可以是 'value' 或 'text'，默认是 'value'。

    • 例子： setSelects-getSelects

    setSelects

    • 参数： values

    • 说明： 设置选定的值。values 的类型为数组。

    • 例子： setSelects-getSelects

    enable

    • 参数： -

    • 说明： 启用 multiple select。

    • 例子： enable-disable

    disable

    • 参数： -

    • 说明： 禁用 multiple select。在此模式下，用户不允许操作。

    • 例子： enable-disable

    open

    • 参数： -

    • 说明： 打开 multiple select。

    • 例子： open-close

    close

    • 参数： -

    • 说明： 关闭 multiple select。

    • 例子： open-close

    checkAll

    • 参数： -

    • 说明： 选中所有复选框。

    • 例子： checkAll-uncheckAll

    uncheckAll

    • 参数： -

    • 说明： 取消选中所有复选框。

    • 例子： checkAll-uncheckAll

    focus

    • 参数： -

    • 说明： 使 multiple select 获得焦点。

    • 例子： focus-blur

    blur

    • 参数： -

    • 说明： 使 multiple select 失去焦点。

    • 例子： focus-blur

    refresh

    • 参数： -

    • 说明： 重新加载 multiple select。如果您通过 AJAX 或 DOM 操作方法在原始选择上动态添加/删除选项标记，请调用 refresh 进行更新。

    • 例子： refresh

    destroy

    • 参数： -

    • 说明： 销毁 Multiple Select。如果你想删除 multiple select 的属性。

    • 例子： destroy

    多语言

    多语言的使用语法：

    $('#multiple').multipleSelect({
  formatMessage: function () {
    return 'Format message'
  }
})

    例子：The i18n

    formatSelectAll

    参数： undefined

    默认： '[Select all]'

    formatAllSelected

    参数： undefined

    默认： 'All selected'

    formatCountSelected

    参数： count, total

    默认： count + ' of ' + total + ' selected'

    formatNoMatchesFound

    参数： undefined

    默认： 'No matches found'
